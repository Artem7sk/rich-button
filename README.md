## 🚀 О проекте
**Richest Button** — это не просто бот. Это закрытый цифровой клуб, развернутый на частном сервере **MARVEL**. Здесь нет места случайным людям. Входной билет — **50 Telegram Stars**, которые подтверждают твой статус и серьезность намерений.

Каждое нажатие — это твой вклад в глобальную экосистему и твоё место в мировом рейтинге богачей.

## ✨ Фишки клуба
* **Lux-Интерфейс:** Золотая 3D-кнопка с тактильной отдачей (Haptic Feedback) и звуком золотых монет.
* **Прозрачный Топ:** Рейтинг лидеров обновляется в реальном времени.
* **Система Рангов:** Пройди путь от *Прохожего* до *Владельца MARVEL*.
* **Технологии:** Python (Aiogram 3), Docker, SQLite, и GitHub Pages для фронтенда.
